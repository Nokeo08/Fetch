# Items and Sections

## Sections

Sections organize items within a list by category (e.g., "Produce", "Dairy", "Frozen").

### Creating a Section

1. Open a list
2. Add a new section with a name (1-100 characters)
3. Sections are added at the end of the list

### Reordering Sections

Drag sections to reorder them within a list.

### Editing a Section

Tap a section name to rename it. The change syncs to all devices immediately.

### Deleting a Section

Deleting a section removes the section and all items in it. This cannot be undone.

## Items

Items represent individual things to buy or collect.

### Creating an Item

1. Open a section within a list
2. Add an item with:
   - **Name** (required, 1-200 characters)
   - **Quantity** (optional, e.g., "2 lbs", "6 pack")
   - **Description** (optional, additional notes)
3. As you type, autocomplete suggests items from your history

### Item States

Each item has one of three states:

| State | Meaning |
|-------|---------|
| **Active** | Not yet acquired -- the default state |
| **Completed** | Acquired or done -- visually marked as complete |
| **Uncertain** | Unsure if needed -- flagged for review |

Tap an item to change its state. Completed items are sorted to the bottom of their section.

### Editing an Item

Update any field (name, quantity, description, status) by tapping on the item. All changes sync in real-time.

### Moving Items Between Sections

Items can be moved from one section to another within the same list. The item is placed at the end of the target section.

### Reordering Items

Drag items to reorder them within a section. Non-completed items appear first, followed by completed items.

### Deleting an Item

Delete individual items by tapping the delete action. This cannot be undone.

## Item History and Suggestions

Fetch automatically remembers items you've added. This history powers the autocomplete suggestions:

- Start typing an item name to see suggestions
- Suggestions are ranked by how frequently you've used them
- Fuzzy matching finds items even with typos (minimum 2 characters)
- History entries include the last-used section name, description, and quantity

You can manage your item history from Settings to remove entries you no longer need.

## Sorting Behavior

Within a section, items are displayed in this order:

1. **Non-completed items** -- sorted by their drag-and-drop position
2. **Completed items** -- sorted to the bottom
