# Managing Lists

Lists are the top-level organizer in Fetch. Each list contains sections, and each section contains items.

## Creating a List

1. From the Lists page, tap the add button
2. Enter a name (1-100 characters, must be unique)
3. Optionally choose an icon (defaults to the clipboard icon)
4. The new list is automatically set as the **active list**

List names are case-insensitive unique -- you cannot have both "Groceries" and "groceries".

## Active List

One list can be marked as the **active list** at a time. The active list is the one shown by default when you open Fetch. To change the active list, tap the activate button on any list.

## Editing a List

You can update a list's name and icon at any time. Changes sync in real-time to all connected devices.

## Reordering Lists

Drag lists to reorder them. The order is preserved across sessions.

## Deleting a List

Deleting a list permanently removes the list and all of its sections and items. This action cannot be undone.

## Lists and Sections

Each list contains one or more **sections** that help organize items by category. See [Items and Sections](items.md) for details on managing sections and items within a list.

## Creating a Template from a List

You can save any list as a reusable template:

1. Open the list
2. Use the "Save as Template" option
3. Enter a template name
4. Optionally select which sections to include

See [Templates](templates.md) for more on working with templates.

## Real-Time Sync

All list operations (create, edit, delete, reorder, activate) are broadcast to all connected devices via WebSocket. Changes appear instantly without refreshing.
